NSW Weather Forecast Data

Weather forecasts for selected NSW locations from the Australian Bureau of Meteorology.

Download

forecast.csv - Today's weather + 7-day forecast

Data Format

location,today_min,today_max,day1_min,day1_max,...,day6_min,day6_max,rain_mm,fire_danger
Lithgow,12,18,12,16,11,18,11,19,9,22,9,26,11,26,1-30,Moderate

Fields

  • location - Location name
  • today_min, today_max - Today's min/max temperature (Celsius)
  • day1_min, day1_max ... day6_min, day6_max - Following 6 days min/max temperatures
  • rain_mm - Estimated rainfall range for today (e.g., "0-5" or "0" if none)
  • fire_danger - Fire danger rating (e.g., "Moderate", "High", "Extreme")

Source

Data sourced from the Bureau of Meteorology API.

Updated daily via GitHub Actions.
