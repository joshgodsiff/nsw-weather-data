NSW Weather Forecast Data

Daily temperature forecasts for NSW locations from the Australian Bureau of Meteorology.

Download

forecast.csv - Nearest min/max temperatures for all NSW locations

Data Format

aac,location,min_temp_celsius,min_temp_date,max_temp_celsius,max_temp_date
NSW_PT131,Sydney,21,2026-01-09,33,2026-01-08
...

Fields

  • aac - BOM area code
  • location - Location name
  • min_temp_celsius - Nearest forecast minimum temperature
  • min_temp_date - Date the min temp applies to
  • max_temp_celsius - Nearest forecast maximum temperature
  • max_temp_date - Date the max temp applies to

Source

Data sourced from Bureau of Meteorology (IDN11060).

Updated daily via GitHub Actions.
