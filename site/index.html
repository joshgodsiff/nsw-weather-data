NSW Weather Forecast Data

Weather forecasts for selected NSW locations from the Australian Bureau of Meteorology.

Download

forecast.csv - Today's weather + 7-day forecast

Data Format

location,state,postcode,geohash,today_min_c,today_max_c,fire_danger,rain_min_mm,rain_max_mm,rain_chance_pct,day0_date,day0_min_c,day0_max_c,...
Lithgow,NSW,2790,r64c839,12,18,Moderate,10,30,90,2026-01-16,,18,...

Fields

Location Info

  • location - Location name
  • state - State code
  • postcode - Postal code
  • geohash - BOM geohash identifier

Today's Weather

  • today_min_c - Today's minimum temperature (Celsius)
  • today_max_c - Today's maximum temperature (Celsius)
  • fire_danger - Fire danger rating (e.g., "Moderate", "High", "Extreme")
  • rain_min_mm - Minimum expected rainfall (mm)
  • rain_max_mm - Maximum expected rainfall (mm)
  • rain_chance_pct - Chance of rain (%)

7-Day Forecast

For each day (0-6), where day 0 is today:

  • dayN_date - Forecast date (YYYY-MM-DD)
  • dayN_min_c - Minimum temperature (Celsius)
  • dayN_max_c - Maximum temperature (Celsius)

Source

Data sourced from the Bureau of Meteorology API.

Updated daily via GitHub Actions.
